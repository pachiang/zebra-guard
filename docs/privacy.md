# 隱私聲明

> 本文件對使用者說明 ZebraGuard 如何處理您的資料。草稿階段,正式發佈前應由有專業資格者審閱。

## 一、核心承諾

**您的影片與分析結果,完全保存在您自己的電腦中。**

- ZebraGuard 不會將您的影片、截圖、或任何分析結果上傳至任何伺服器
- ZebraGuard 可在完全離線的情況下執行
- ZebraGuard 不蒐集您的影像內容、GPS 資訊、或任何個人身分識別資料

## 二、會連線到網路的情境

下列情境**且僅限於下列情境**,ZebraGuard 會發出網路請求。任何一項都不會上傳您的影片或分析結果。

1. **首次啟動下載模型**:從 GitHub Release 下載 AI 模型權重檔,這是單純的檔案下載,僅傳出您的 IP 位址(GitHub 會記錄)。
2. **檢查更新**(Phase 2 功能,尚未實作):若您手動點選「檢查更新」,本程式會查詢本專案的 GitHub Release 頁面取得最新版號。
3. **當機回報**(Phase 2 功能,尚未實作;且為 opt-in):僅在您主動開啟此選項的情況下,程式當機時會傳送錯誤堆疊到 Sentry(或類似服務)。**絕不包含**您的影片內容或路徑。

## 三、本機儲存

本程式會在您的電腦建立以下資料:

| 資料 | 位置 | 內容 |
|---|---|---|
| 程式設定 | `%APPDATA%\ZebraGuard\` | 介面偏好、最近開啟專案、模型路徑 |
| AI 模型 | `%LOCALAPPDATA%\ZebraGuard\models\` | 下載的模型權重檔 |
| 日誌 | `%LOCALAPPDATA%\ZebraGuard\logs\` | 程式運作日誌(不含影像內容,僅含時間、錯誤訊息) |
| 專案檔 | 您選擇的位置 | 您的分析結果 |

解除安裝時您可以選擇是否保留專案檔。模型、設定、日誌會被清除。

## 四、影片內可能含有的個人資料

您所處理的影片可能包含路人臉部、車牌、或可識別的個人資訊。**這些資料的蒐集、使用、提供,受《個人資料保護法》規範**。

本程式僅協助您在**您合法取得**的影片中識別違規事件。使用者負有確認影片取得來源合法性、以及後續使用行為合法性的責任。若您打算將匯出的片段用於公開張貼、社群散佈等目的,請先確認是否有合法依據。僅用於向主管機關檢舉之目的,依現行法令通常屬合理使用範圍(但請以現行法為準)。

## 五、第三方元件

本程式內建以下第三方元件,各自有其授權與資料處理方式:

- **PySide6** (Qt for Python) — LGPL / 商業雙授權
- **Ultralytics YOLO** — AGPL-3.0
- **ONNX Runtime** — MIT
- **OpenCV / PyAV** — 各自授權
- **FFmpeg** — LGPL / GPL(視 build 組態)

以上元件皆在您本機執行,不傳送資料至其維護者。

## 六、變更

本聲明若有變更,會在新版本 release note 中說明。您若不同意新版隱私聲明,可停留在舊版或解除安裝。
