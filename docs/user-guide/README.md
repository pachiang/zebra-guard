# 使用手冊

> 本手冊面向**終端使用者**。撰寫中,待 MVP 功能成形後補齊畫面截圖與步驟。

## 規劃內容

- [ ] 安裝與首次啟動
- [ ] 建立新專案:匯入影片
- [ ] 繪製斑馬線 ROI
- [ ] Homography 校正(如何選地面參考點、怎麼量實際距離)
- [ ] 開始分析:GPU / CPU 模式差異、預估時間
- [ ] 審查違規候選:採用 / 拒絕 / 調整時段
- [ ] 匯出檢舉輔助包
- [ ] 如何使用輔助包向警政署提報
- [ ] 常見問題
  - Defender / SmartScreen 警告怎麼辦
  - 分析很慢怎麼辦
  - 為什麼我的車牌/行人沒被偵測到
  - 檔案存在哪裡
  - 怎麼刪除我的資料

## 撰寫原則

- 用語以使用者角度,避免技術名詞(homography → 稱「視角校正」)
- 截圖優先於文字
- 每節目標限於 3 分鐘內可看完
- 明確區分「必要步驟」與「進階調整」
